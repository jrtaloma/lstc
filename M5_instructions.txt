Preparing the M5 dataset

Download the zip file from https://www.kaggle.com/competitions/m5-forecasting-accuracy/data

$ unzip m5-forecasting-accuracy.zip -d datasets
$ rm -rf datasets/__MACOSX
$ rm -rf datasets/m5-forecasting-accuracy/sample_submission.csv
$ python M5_EDA.py
